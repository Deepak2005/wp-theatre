\WPT_Event

Usage:

$event = new WPT_Event(); $event = new WPT_Event($post_id); $event = new WPT_Event($post);

echo $event->html(); // output the details of an event as HTML

echo $event->prices( array('summary'=>true) ) // // a summary of all available ticketprices echo $event->datetime() // timestamp of the event echo $event->date() // localized and formatted date of the event echo $event->time() // localized and formatted time of the event
